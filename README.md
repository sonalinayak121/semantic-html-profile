# Semantic HTML Personal Profile

This project is a **personal profile webpage** created using **semantic HTML5 elements** as part of a learning task.

## Project Objective
The objective of this project is to understand and implement:
- Semantic HTML elements
- Proper document structure
- Accessibility best practices
- Clean and readable HTML code

## Technologies Used
- HTML5 (Semantic Elements)

## Features
- Uses semantic tags such as `header`, `nav`, `main`, `section`, `article`, and `footer`
- Internal navigation using anchor links
- Accessible image using `alt` attribute
- Clear heading hierarchy
- Beginner-friendly, clean HTML structure

## Academic Context
This project was completed as part of my learning journey as a **final-year diploma student**, focusing on building strong fundamentals in web development.

## How to Run the Project
1. Clone or download the repository
2. Open `index.html` in any modern web browser (Chrome, Edge, Firefox)
